Troubleshooting

Common messages

  • “No Twelve Data or Alpha Vantage keys found.”
    Enter valid keys in Settings and click Save Keys.
  • “Alpha Vantage has no 'H4' intraday…”
    Use H1 or D, or rely on Twelve Data for H4.
  • “COT is disabled.”
    Tick Enable COT to show weekly COT details for supported FX pairs.
  • “COT: stale … ignored”
    The latest COT report is older than 35 days; the app hides it to avoid confusion.

Show logs / data

Use Help → Open Logs Folder and Help → Open Data Folder. Include logs when contacting support.

Reset preferences

  1. Close the app.
  2. Delete the app’s data folder shown by Open Data Folder.
  3. Re-open the app and re-enter keys.

Advanced

  • Open DevTools: Ctrl+Shift+I (Mac: Cmd+Option+I).
  • Check the Console for errors (e.g., network, API limits).
