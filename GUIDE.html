1) Setting Up API Keys

SentiGrid pulls market data from public providers. You can use either or both:

Twelve Data

  1. Go to twelvedata.com and create a free account.
  2. Copy your API key.
  3. In SentiGrid, open the API Keys panel and paste it into Twelve Data, then click Save Keys.

Alpha Vantage

  1. Go to alphavantage.co and create a free account.
  2. Copy your API key.
  3. In SentiGrid, paste it into Alpha Vantage, then click Save Keys.

Tip: If you add both, the app can fall back to the other source if one is rate-limited or slow.

2) Loading a Chart

  1. Type a symbol, e.g. EUR/USD (forex) or AAPL (stock).
  2. Choose timeframe: D (daily), H4, or H1.
  3. Click Refresh.
  4. Candlesticks and overlays will render; click legend items to show/hide overlays.
If data doesn’t load: verify keys, try the pair as EURUSD vs EUR/USD, or switch timeframe and refresh again.

4) Risk & Position Size

  • Enter Account Size (e.g., 10000).
  • Enter Risk % per idea (e.g., 1 = 1%).
  • Choose stop method:
    • ATR × multiple (volatility-based), or
    • Custom Stop % (manual).
  • The app calculates an educational position size for comparison.

This is not advice — it’s a learning aid to see how risk changes sizing.

5) Sentiment & COT

  • FXSSI Current Ratio: shows a snapshot of retail long vs short (%). Extreme one-sidedness can be informative.
  • COT (Commitment of Traders): weekly “leveraged funds” positioning (net = long − short). Major futures like EUR, JPY, GBP are best-covered.
COT updates weekly (usually Friday for Tuesday’s positions). Some markets can be delayed; if unavailable, treat COT as “off” for that pair.

7) Filters & Confirmation

  • MTF Confirmation: combine a higher-timeframe view with a lower-timeframe check.
  • 200-EMA Trend Filter: restricts ideas to trend direction only (Sell-only or Buy-only toggles).
  • EMA Momentum (14/50): basic momentum check (faster vs slower EMA).
  • Volatility Filter & Volatility Sizing: avoid low-quality conditions or scale size down in high vol.
  • Apply Sentiment Sizing: if retail sentiment is ≥ a threshold (e.g., 70% one-sided), size down (e.g., 50%).

8) Troubleshooting

  • Check the Status panel at the bottom of the app. It shows what the app is doing (key loads, data requests, errors) in time order.
  • Under Chart, read the Data source line (e.g., “Twelve Data (H4)” or “Alpha Vantage (Daily)”) to confirm what actually loaded.
  • Keys: In API Keys, click Test Load Keys to verify they’re saved. Use Save Keys if needed, or Delete Saved Keys to reset and re-enter.
  • Timeframe: If H4 won’t load with Alpha Vantage, switch to H1 or D, or rely on Twelve Data for H4.
  • Symbols: try both formats (EUR/USD vs EURUSD), then press Refresh.
  • Rate limits / outages: free tiers can throttle. Try again later or add the second provider’s key as backup.
  • Closed markets: weekends/holidays may show gaps or no new bars—this is normal.

When contacting support: copy the last lines from Status, include your symbol, timeframe, local time & timezone, and which provider(s) you use. Email: sentigrid.help@gmail.com.

9) Keyboard Shortcuts

  • Ctrl/Cmd + R — Refresh data
  • Ctrl/Cmd + L — Focus Symbol box
  • Ctrl/Cmd + + / − — Zoom UI (if enabled)
  • Esc — Close dialogs

Windows/Linux use Ctrl; macOS uses Cmd.

10) FAQ

Do I need both API keys?
No. One key is enough. Adding both improves reliability.

Why do candles sometimes stop?
Free tiers can throttle. Try again later or refresh with a different timeframe.

Why does COT show “unavailable”?
Some instruments are not mapped or are delayed. Use pairs backed by major futures (EUR, JPY, GBP, CAD, AUD, CHF).
