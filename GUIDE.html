Sentigrid — User Guide

Welcome! This guide helps you set up and use the Sentigrid Trading Dashboard.

1) First run & legal consent

On first launch you’ll see a consent dialog (not investment advice, adult confirmation, EU waiver). You must accept to proceed.

2) Add data keys

  1. Open the app.
  2. Enter your Twelve Data and/or Alpha Vantage API keys in the Settings panel.
  3. Click Save Keys. You should see “Saved keys” in the status log.

3) Pick symbol & timeframe

  • Type a symbol (e.g., EUR/USD or AAPL).
  • Choose granularity: D, H4, or H1.
  • Click Refresh to draw the chart.
Note: Alpha Vantage doesn’t provide H4; the app automatically uses Twelve Data for H4 when available.

4) Reading the chart

  • Main candles (with OHLC tooltip)
  • Overlays: EMA, Bollinger Bands; ATR on the right axis
  • Hover to see values; intraday charts drop the still-forming last bar

5) Signals & filters

The “Example (not advice)” line summarizes a simple rule using EMA + Bollinger midline. Optional filters:

  • 200-EMA Trend Filter — blocks counter-trend signals
  • Momentum (EMA14 vs EMA50) — blocks weak momentum
  • MTF confirmation — checks a lower timeframe for agreement

6) Risk & position sizing

  • Choose Stop method: ATR × multiple or Custom %.
  • Set account size and risk % (per trade) to see position notional & units.
  • Take-profit (TP) uses your risk:reward ratio (e.g., 2:1).

7) COT (Commitments of Traders)

  • Enable COT to view weekly net positions (Leveraged Funds) for major FX.
  • If a report is stale (>35 days), the app hides it automatically.

8) Sentiment

Optionally aggregates external sentiment (e.g., FXSSI). You can size down positions if the crowd is extreme.

9) Troubleshooting basics

  • Check the Status Log at the bottom for messages.
  • Open the Help → Troubleshooting page for solutions.
  • Use Ctrl+Shift+I (or Cmd+Option+I on Mac) to open DevTools and see errors.
